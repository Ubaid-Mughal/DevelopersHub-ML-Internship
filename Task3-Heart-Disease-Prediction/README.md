&#x20;**Task 3 - Heart Disease Prediction**

Objective: Predict whether a patient has heart disease using health data



Dataset: Heart Disease UCI Dataset (920 rows, 16 columns)



Model: Logistic Regression



**Results:**

\- Accuracy: 80%

\- AUC: 0.89

\- Strongest predictors: oldpeak, cp\_asymptomatic, exang, thalch

\- Weakest predictor: cholesterol (despite common belief)
